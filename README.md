## Setup
### 1. Clone the repository first using this command
```
git clone https://github.com/sukhmani1303/acm-devcup-minerva-2022.git
```

### 2. Install dependencies 
```
pip install -r requirements.txt
```

## How to run

Run in terminal
```
python main.py
```

Open browser and type
```
http://127.0.0.1:8000/
```
